## 🔄 数据流优化

### 优化前（重复读取）：
```
PM模块 ──→ 读取所有文件（第1次）
   ↓
Planner模块 ──→ 读取所有文件（第2次）
   ↓
Preprocessor模块 ──→ 读取所有文件（第3次）
```

### 优化后（数据传递）：
```
PM模块 ──→ 读取文件元数据（快速）
   ↓         ↓
   ↓      file_data（传递）
   ↓         ↓
Planner模块 ←┘ 读取完整内容（仅1次）
   ↓         ↓
   ↓      file_data（传递）
   ↓         ↓
Preprocessor模块 ←┘ 复用数据
```
